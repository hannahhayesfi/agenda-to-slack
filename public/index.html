Agenda Slack Notification Service

This is a secure proxy service for sending notifications to Slack when users visit the thank you page.

Available API Endpoints:

/api/notify-visit

Sends a notification to Slack when a user visits the thank you page.

/api/test

Test endpoint to verify the API is working correctly.

Implementation:

To use this service, add the following script to your thank you page:


// Only run on the thank you page
if (window.location.pathname.includes('/thank-you')) {
  // Get referrer information
  const referrer = document.referrer || 'Direct visit';
  
  // Send notification to the API endpoint
  fetch('https://agenda-to-slack-2j2w.vercel.app/api/notify-visit', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      referrer: referrer,
      page: window.location.href,
      timestamp: new Date().toISOString()
    }),
  })
  .then(response => {
    if (response.ok) {
      console.log('Notification sent successfully');
    }
  })
  .catch(error => {
    // Silent fail
    console.error('Error sending notification:', error);
  });
}
